Dependencies
============
 Security & Auth: npm install bcrypt bcryptjs jsonwebtoken jwt-decode
 Server & Middleware: npm install express cors helmet cookie-parser
 Utilities: npm install dotenv joi init
 Encryption: npm install encryptor simple-encryptor
 Email: npm install nodemailer
 Database: npm install mongoose
 Development Tool (Auto Restart): npm install --save-dev nodemon
